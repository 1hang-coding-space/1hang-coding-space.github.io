BFS

核心数据结构为队列，基本操作为元素出队列，将其基础上的“趋向目标”元素入队列，循环重复直至满足条件或队列空。

例1：哈密顿圈问题

在多面体或平面图中，从一个顶点开始沿着棱（边）走遍所有其他顶点回到原地，且每一顶点只通过一次。这样的路径就是一条哈密顿回路。求证正十二面体存在哈密顿回路。

解：易知正十二面体的每个面是正五边形，绘制其平面展开图，对每个节点标号，并用邻接表的形式存储图，进行广度优先搜索，若路径的当前节点不是“死节点”（相邻节点都已在路径中），则将新路径入队列。

例2：二叉树左小角的值
LeetCode513，链接：https://leetcode.cn/problems/find-bottom-left-tree-value/

找出二叉树最底层最左边节点的值。

解：用first变量记录每一层最左边的节点，若遍历完最后一层，队列变空，退出循环。

例3：最小基因变化
LeetCode433，链接：https://leetcode.cn/problems/minimum-genetic-mutation

基因序列可以表示为一条由 8 个字符组成的字符串，其中每个字符都是 ‘A’、’C’、’G’ 和 ‘T’ 之一。
假设我们需要调查从基因序列 start 变为 end 所发生的基因变化。一次基因变化就意味着这个基因序列中的一个字符发生了变化。
例如，”AACCGGTT” –> “AACCGGTA” 就是一次基因变化。
另有一个基因库 bank 记录了所有有效的基因变化，只有基因库中的基因才是有效的基因序列。
给你两个基因序列 start 和 end ，以及一个基因库 bank ，请你找出并返回能够使 start 变化为 end 所需的最少变化次数。如果无法完成此基因变化，返回 -1 。
注意：起始基因序列 start 默认是有效的，但是它并不一定会出现在基因库中。

输入：start = “AACCGGTT”, end = “AAACGGTA”, bank = [“AACCGGTA”,”AACCGCTA”,”AAACGGTA”]
输出：2

解法1：初始队列为[start]，对于队列中的每个基因序列探寻其“最小变化”是否合法（在bank中且不在visited中）或等于end，将合法者入队列，等于end则return step，将检验元素出队列；一轮结束后step++，重复以上过程，若最后队列空则return -1。

tips：js中不要在forEach循环中return函数结果，因为forEach中的return不会终止循环

解法2：在解法1的基础上做预处理优化，用邻接表存储bank中各个基因序列的两两相邻状况；初始队列为与start相邻且在bank中的基因序列，然后对其进行类似解法1的操作，优化点在于不需要对队列中的每个基因序列的每个元素依次做替换，只需到邻接表中读取相应数据即可。

例3：出现次数最多的子树元素和
LeetCode508，链接：https://leetcode.cn/problems/most-frequent-subtree-sum/

给出一个二叉树的根结点 root ，返回出现次数最多的子树元素和。如果有多个元素出现的次数相同，返回所有出现次数最多的子树元素和（不限顺序）。
一个结点的 「子树元素和」 定义为以该结点为根的二叉树上所有结点的元素之和（包括结点本身）。

输入: root = [5,2,-3]
输出: [2,-3,4]

输入: root = [5,2,-5]
输出: [2]

解：用一个哈希表存储子树元素和以及它出现的次数，利用广度优先搜索更新哈希表。

2-pointers

例：找出与数组相加的整数
LeetCode508，链接：https://leetcode.cn/problems/find-the-integer-added-to-array-ii/description/

给你两个整数数组 nums1 和 nums2。
从 nums1 中移除两个元素，并且所有其他元素都与变量 x 所表示的整数相加。如果 x 为负数，则表现为元素值的减少。
执行上述操作后，nums1 和 nums2 相等 。当两个数组中包含相同的整数，并且这些整数出现的频次相同时，两个数组相等 。
返回能够实现数组相等的最小整数 x 。

解：
对两个数组分别排序后，转化为子序列问题。由于nums1长度比nums2多2，因此在nums1排序后的前三个元素中，必有一个是序列的起点
又因为题目要求返回满足要求的最小整数，所以从nums1[2]开始往前讨论至nums1[0]
应用双指针来判断nums2是否为nums1的子序列
对长的序列用for循环进行遍历，对短的序列用指针记录位置，满足要求则+1，若行至最后一位，则符合条件
