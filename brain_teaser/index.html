find-all-duplicates-in-an-array

LeetCode442，链接：https://leetcode.cn/problems/find-all-duplicates-in-an-array/

给出一个为n的整数数组nums，其中nums的所有整数都在范围 [1, n] 内，且每个整数出现一次或两次。试找出所有出现两次的整数，并以数组形式返回。
要求时间复杂度为 O(n) 且仅使用常量额外空间的算法解决此问题。

输入：nums = [4,3,2,7,8,2,3,1]
输出：[2,3]

解：尽量使数组中的第i个数就是i，最后不满足的此规律的位置就是重复数字的位置。实现方法为遍历数组，重复将当前位置上的数字k与数组的第k个数字交换，直至两者相等。

minimum-moves-to-equal-array-elements-ii

LeetCode462，链接：https://leetcode.cn/problems/minimum-moves-to-equal-array-elements-ii/

给定一个长度为 n 的整数数组 nums ，返回使所有数组元素相等需要的最少移动数。
在一步操作中，你可以使数组中的一个元素加 1 或者减 1 。

输入：nums = [1,2,3]
输出：2
解释：
只需要两步操作（每步操作指南使一个元素加 1 或减 1）：
[1,2,3] => [2,2,3] => [2,2,2]

解：目标数是数组的中位数。若对数组排序，从左右两端开始依次取上下界，当且仅当目标数落在范围满足“最小移动次数”，不断逼近直至中位数（若数组长度为奇数）或中间两个数之间任意数（若数组长度为偶数）

tips：js的sort()函数如果不传参，默认为ASCII码非降序排序，若遇到多位数排序很可能无意义；上述代码使用nums.sort((a,b) => a - b)，第一个参数是数组后一个数，第二个参数是数组前一个数，返回值是负数则交换顺序。因此nums.sort((a,b) => a - b)代表非降序排序。

container-with-most-water

LeetCode11，链接：https://leetcode.cn/problems/container-with-most-water/

数组中 min(nums[i], nums[j]) * (j - i) 的最大值。

解：采用双指针的思路，每次移动指向较小的元素的指针。时间复杂度由O(n^2)降为O(n)。
